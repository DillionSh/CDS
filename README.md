# Mental Health Risk Predictor

A Streamlit web app that predicts student mental health risk using Logistic Regression, Random Forest, XGBoost, and Neural Network models.

## Setup

1. Install dependencies:
   ```bash
   python -m pip install streamlit pandas scikit-learn xgboost numpy

2. Run the app:
streamlit run main.py

3. Open your browser at http://localhost:8501
